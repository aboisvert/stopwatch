Stopwatch API to easily monitor production applications.

This API can be used to determine application performance bottlenecks, user/application interactions, and application scalability. Each stopwatch gathers summary statistics such as hits, execution times (total, average, minimum, maximum, standard deviation), and simultaneous application requests.

Basic Usage

    import stopwatch.Stopwatch
    import stopwatch.StopwatchFactory
    import stopwatch.StopwatchStats

    Stopwatch("test") {
      // code being timed
    }

Usage for non-inlined code

    val stopwatch = Stopwatch.start("test" )

    // and later, elsewhere in your code
    stopwatch.stop()
    
    // or
    Stopwatch.stop("test")

Displaying statistics

    // show statistics
    stats = Stopwatch.snapshot("test")
    Console.println(stats)
